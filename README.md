# Clock - JS Challenge

Este proyecto consiste en un reloj analógico interactivo construido con tecnologías web básicas. La lógica se encarga de calcular y rotar las manecillas de horas, minutos y segundos en tiempo real.

## Tecnologías utilizadas

  * **HTML5**: Estructura semántica del reloj.
  * **CSS3**: Estilos para el diseño del reloj, uso de Flexbox para el centrado, transformaciones (`rotate`, `transform-origin`) y transiciones para el movimiento fluido de las manecillas.
  * **JavaScript (ES6)**: Lógica para obtener la hora actual mediante el objeto `Date` y manipulación del DOM para actualizar la posición de las manecillas cada segundo.

## Instrucciones para instalar o ejecutar localmente

Para ejecutar este proyecto en tu máquina local, sigue estos pasos:

1.  **Clonar el repositorio**:
    ```bash
    git clone https://github.com/arieloou/js-challenge.git
    ```
2.  **Navegar a la carpeta del proyecto**:
    ```bash
    cd js-challenge
    ```
3.  **Abrir el archivo**:
    Simplemente abre el archivo `index.html` en tu navegador web de preferencia (Chrome, Firefox, Edge, etc.). No es necesaria ninguna instalación de dependencias ni servidores adicionales.
